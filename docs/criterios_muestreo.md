# Criterios de uso y muestreo

## Datos maternos

Registrar edad, presión sistólica, presión diastólica, glucosa, temperatura y frecuencia cardiaca materna.

## Datos fetales

El modelo fetal requiere 21 variables CTG procesadas. No deben ser inventadas manualmente en un uso real.

## Edad gestacional

La edad gestacional se usa como contexto de seguridad, no como variable directa del modelo fetal ni materno entrenado.

## Señal fetal

Antes de interpretar la frecuencia cardiaca fetal, debe confirmarse que el latido registrado corresponde al feto y que la señal es clara.
